Proyectos Personales

ZeroSteg (HTML, CSS, JavaScript)

Esteganografía Sitio de web interactivo Unicode y caracteres de cero anchura

Un programa de esteganografía basado en el web, para cifrar y decifrar mensajes de texto adentro de un otro texto. ZeroSteg usa los caracteres “joiners” y “non-joiners” con cero anchura en Unicode. El enlace en el título va directamente al sitio web de ZeroSteg. Si quiere ver el repositorio del proyecto incluyendo el código fuente, haga click aquí.

Rosalind (Python)

Bioinformática Biología computacional Estructuras de datos Algoritmos Técnicas de resolución de problemas Relaciones de recurrencia Teoría de grafos Programación dinámica Combinatoria y probabilidad Secuencias y series

Rosalind (Enlace al sitio web) es una plataforma para problemas y algoritmos de programación bioinformática. Es un proyecto conjunto entre la Universidad de California en San Diego, Universidad Académica de San Petersburgo, y la Academia Rusa de Ciencias. Mi cuenta GitHub contiene mis soluciones propias a los problemas de Rosalind, implementadas en Python y listadas por ID del problema. A partir del 7 de agosto 2019, estoy en el 1% superior de usuarios por número de problemas resueltos.

BitmapParser (C++)

Librería Entrada y salida de archivos Procesamiento de imágenes Estructuras de datos Manejo de errores Programación orientada a objetos

Una librería corta para leer y escribir imágenes bitmap. Contiene funciones básicas para editar imágenes (reflejar, transponer, girar, cortar, sobreponer, filtros de color). Conveniente para extraer píxeles y sus valores RGB como vectores de estructuras.

EasyLSB (C++)

Esteganografía Bit menos significativo Manipulación de bits Manejo de errores Herencia Iterador personalizado

Un programa para cifrar y decifrar mensajes de texto adentro de imágenes bitmap usando esteganografía del bit menos significativo. Resulta en poco o no diferencia visible en la imagen. Incluye mi librería BitmapParser.
