개인 프로젝트

BitmapParser (C++)

라이브러리 파일 입출력 이미지 편집 자료 구조 오류 처리 객체 지향 프로그래밍

비트맵 이미지를 읽고 쓰기 위한 작은 라이브러리입니다. 대칭, 전치, 회전, 자르기, 겹치기, 색 필터 등의 간단한 편집 기능도 지원합니다. 이미지에서 픽셀과 RGB 색상 값을 행렬로 출력할 때 유용합니다.

EasyLSB (C++)

스테가노그래피 최하위 비트 비트 조작 오류 처리 객체 상속 맞춤형 반복자

최하위 비트 스테가노그래피를 사용하여 비트맵 이미지 속 메시지를 읽고 쓰는 프로그램입니다. 숨겨진 메시지가 담긴 이미지는 육안으로 볼 때 원본과 거의 차이가 없습니다. 이 프로그램은 저의 BitmapParser 라이브러리를 포함합니다.

jasonkimprojects.github.io (HTML, CSS, Bootstrap)

본 웹사이트. 저의 이력서 및 GitHub와 LinkedIn 프로파일과 연결된 링크가 있습니다. 저의 학업과 개인 프로젝트를 요약한 정보를 담고 있습니다.

Evo(lved) (Android)

맞춤형 펌웨어 스크립트 리눅스 최적화

2048 (Java, Swing)

게임 행렬 작업 GUI 프로그래밍 리스너

고등학교 AP Computer Science 최종 프로젝트의 첫번째 부작이었습니다. 처음부터 시작하여 원본 게임의 레플리카를 구현하였습니다. Swing 위젯 툴킷을 사용한 그래픽, 그리고 키보드 입력을 주시하는 리스너를 연습할 기회였습니다.

Pong (Java, Swing)

게임 GUI 프로그래밍 리스너

고등학교 AP Computer Science 최종 프로젝트의 두번째 부작이었습니다. 이 역시 처음부터 시작하여 원본 게임의 레플리카를 구현하였고, 그래픽 기반 게임과 키보드 입력을 처리하는 또 하나의 연습이었습니다.

Blackjack (Java)

또 하나의 고등학교 AP Computer Science 프로젝트였으며, 제가 처음부터 만든 첫 번째 게임이었습니다. 명령 프롬프트를 통해 게임을 할 수 있으며, 여러 명의 인간 플레이어와 기초적인 컴퓨터 플레이어를 지원합니다.
